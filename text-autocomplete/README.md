# Проект "Автодополнение текстов"

## Данные
Использован датасет коротких текстовых постов (Sentiment140).
Выполнена очистка и нормализация текста (lowercase, удаление ссылок и шумовых символов), токенизация и формирование обучающих примеров.

## Задача
Реализовать систему автодополнения текста, которая по префиксу предсказывает продолжение.  
Задача формализована как итеративное предсказание следующего токена до достижения `<eos>` или лимита длины.

## Описание работы
- Подготовлен пайплайн предобработки данных
- Реализован PyTorch Dataset для next-token prediction
- Построена и обучена лёгкая LSTM-модель
- Реализована генерация текста
- Оценено качество по метрикам ROUGE-1 и ROUGE-2
- Запущена предобученная модель **distilgpt2**
- Проведено сравнение лёгкой RNN и Transformer
- Сформулированы рекомендации с учётом требований к мобильности

## Используемые библиотеки
- *PyTorch*
- *transformers*
- *evaluate (ROUGE)*
- *pandas*
